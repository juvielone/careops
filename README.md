# CareOps — Healthcare Operations & Incident Management System

CareOps is a modern healthcare operations platform designed to help clinics, aged care providers, and healthcare teams efficiently manage incidents, staff coordination, and operational workflows.

This project is built as a portfolio-ready SaaS application to demonstrate real-world full-stack engineering practices, including scalable frontend architecture and backend API integration (in progress).

---

## 🔗 Live Demo

👉 https://careops-khaki.vercel.app/

---

## 📌 Project Overview

CareOps focuses on simplifying operational challenges in healthcare environments by providing:

* Incident tracking and management
* Staff coordination and assignment
* Activity logs and audit trails
* Reporting and operational insights

The goal is to build a system similar to tools used in real healthcare operations, with a strong emphasis on usability, performance, and scalability.

---

## 🚀 Features (Current)

* Responsive dashboard interface
* Incident listing and detail views
* Activity tracking UI
* Modular page structure (React Router)
* Lazy-loaded routes for performance optimization
* Clean UI built with Tailwind CSS

---

## 🛠️ Tech Stack

### Frontend

* React + TypeScript
* Vite (build tool)
* React Router (routing)
* Tailwind CSS (styling)

### Backend (in progress)

* ASP.NET Core Web API
* SQL Server / PostgreSQL (planned)

## ⚙️ Project Structure

## 🧠 Key Engineering Highlights

* Migrated from Next.js to Vite for simpler and faster builds
* Implemented route-based lazy loading to improve performance
* Structured components for scalability and maintainability
* Clean separation between frontend and backend architecture
